# Craftycommerce

i am part of a development team tasked with building "CraftyCommerce", an online marketplace for handmade goods. The project is split into various features, and we will use Git to handle changes and collaboration effectively.



## project overview

1) Feature product listing - display a comprehensive list of products with sorting and filtering option

2) feature shopping cart - enable users to add produucts to their cart and proceed to checkout

3) feature customer reviews - allow customers to leave reviews and ratings to help others to make decision
